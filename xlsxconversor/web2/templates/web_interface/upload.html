Como Funciona?
  1. Seleção do Estado: Escolha o estado de origem da sua empresa
  2. Upload do Excel: Envie seu arquivo BMP em formato .xlsx
  3. Conversão Automática: O sistema converte para XML seguindo o padrão BMPCAMPO.xsd
  4. Assinatura Digital: XML é assinado com certificado A1 ICP-Brasil
  5. Envio SEFAZ: Transmissão segura via webservice para SEFAZ-ES
Requisitos do Arquivo
  • Formato: Excel .xlsx
  • Tamanho máximo: 10MB
  • Estrutura conforme modelo BMP
  • Dados válidos (CNPJ, datas, volumes)
  • Máximo 6500 registros por seção
